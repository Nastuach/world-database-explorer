Описание проекта:
Проект предоставляет удобный интерфейс для выполнения SQL-запросов к базе данных MySQL (world).  
Пользователь может:  
✔ Получать данные о столицах и их языках  
✔ Анализировать страны по форме правления и континенту  
✔ Просматривать суммарную площадь континентов  

Технологии:
- Backend: PHP 7.4+ (PDO для работы с MySQL)  
- Frontend: HTML5, CSS3 (чистый код без фреймворков)  
- База данных: MySQL (используется стандартная БД `world`)  

Запуск:
1. Требования:
- Веб-сервер (Apache/Nginx)  
- PHP 7.4+  
- MySQL 5.7+  
- База данных `world`  
2. Установка:  
1) Склонируйте репозиторий:  
   git clone https://github.com/Nastuach/world-database-explorer.git
2) Импортируйте БД `world.sql` в MySQL  
3) Настройте подключение в `db_connection.php`:  
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'world');
   define('DB_USER', 'ваш_логин');
   define('DB_PASS', 'ваш_пароль');
3. Запуск:
php -S localhost:8000
Откройте в браузере: http://localhost:8000

Функционал
1. Запросы к столицам
- Фильтрация по:  
  - Минимальному населению  
  - Количеству официальных языков  
2. Анализ стран
- Поиск по:  
  - Форме правления (Republic, Monarchy и т.д.)  
  - Континенту  
3. Площади континентов 
- Таблица с суммарной площадью всех стран
